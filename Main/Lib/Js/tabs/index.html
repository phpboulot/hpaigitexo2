L'onglet correspondant à la page active dispose de la classe "selected" pour mettre en relief l'onglet actif.
La solution consiste à associer la classe selected à l'onglet No Script:

<li><a href="demo_apply_noscript.html" class="selected">No Script</a></li>

 

En version 2, l'equipe d'assist developpement assure la compliance du code de la librairie selon les guidelines 2.3 et 6:
  • Checkpoint 3.4, priorité 2:
    • utilisation des unités relatives ainsi que 'em' pour les offsets de positionnement
  • Checkpoint 2.2, priorité 2 (images), 1(text):
    • codage des couleurs par code numérique de préférence au nom générique en particulier dans le css
  • Checkpoints 1.1 et 6.2, priorité 1:
    • Mise en oeuvre de l'élément NOSCRIPT
Ce test unitaire démontre la possibilité de mettre en oeuvre des onglets en utilisant une technique basée sur le css de préférence à l'appel par js.
